User Manual 📘

How to use the dashboard, digit analyzer, and where to find your Deriv API token.

What this dashboard does

  • Connects your Deriv account with your API token
  • Shows live ticks and charts
  • Lets you run the bot on your own private account session
  • Shows P&L, wins, losses, trades, and current stake
  • Shows your access timer if admin set an expiry date

How to find your Deriv API token

  1. Log in to your Deriv account
  2. Open the Deriv API area or token manager
  3. Go to Manage Tokens or API Token Manager
  4. Create a token with the scopes needed for trading and account use
  5. Copy that token and paste it into this dashboard

Use only the permissions you actually need. Test with demo first.

How to start the bot

  1. Paste your Deriv API token
  2. Click Connect
  3. Select a market
  4. Click Start Ticks
  5. Wait for candles to appear
  6. Adjust settings if needed
  7. Click Start Bot

Digit Analyzer modes

  • Observer Analyzer watches last digits, builds samples, calculates bias, and shows signals, but places no digit trades.
  • Executable Analyzer uses the same analysis but is allowed to place DIGITEVEN or DIGITODD trades once the signal becomes strong enough.

Digit Analyzer settings

  • Analyzer Precision lets you choose 100 ticks or 200 ticks.
  • 100 ticks is faster and gives more trade opportunities.
  • 200 ticks is slower but gives more precision and stronger confirmation.
  • Analyzer Sample shows how many ticks have been collected out of the selected target.
  • Analyzer Ready turns to Yes when the selected sample target has been reached.

Reset options

  • Reset Session clears session stats, P&L, recent trades, wins, losses, and trade counters.
  • Reset Analyzer clears digit analyzer memory, including collected digit history and rolling samples, so it can start collecting from zero again.

What the buttons do

  • Connect — connects your Deriv session
  • Disconnect — disconnects your Deriv session
  • Start Ticks — starts live market data
  • Start Bot — starts automated trading logic
  • Stop Bot — stops new automated trades
  • Reset Session — clears session stats and recent trades
  • Reset Analyzer — clears digit analyzer samples and bias history
  • Clear Log — clears the visible log box only

What the stats mean

  • Balance — your Deriv balance from the connected session
  • Session P&L — profit/loss for the current session
  • Peak P&L — highest session P&L reached
  • Move % — chart movement from the first visible candle to the latest candle
  • Access Timer — how long your account access remains
  • Analyzer Sample — how many ticks have been collected for the digit analyzer
  • Analyzer Ready — whether the analyzer has reached the selected sample target

Important safety notes

  • Start on demo first
  • Do not share your Deriv token
  • Use the smallest safe stake while testing
  • If your token is exposed, revoke it and make a new one
